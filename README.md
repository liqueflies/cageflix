# cageflix

> A Nicolas Cage clone for Netflix in Vue

## Project idea

> This is a simple Vue.js project training, covering a browsing list of movies with simple server-side rendering achieved from vue-cli.

## Requirements

> Node >= 6.9.4
> yarn / npm

## Build Setup

``` bash
# install dependencies
yarn install

# build project
yarn build

# run project
node server
```
